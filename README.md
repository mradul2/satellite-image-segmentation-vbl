# Vision Beyond Limits

## Requirements
```bash 
pip install wandb
```

## Installation
```bash
git clone https://github.com/mradul2/vbl.git
```

## Usage

```bash
python3 main.py config.json [--mode MODE] [--model MODEL]
        [--wandb WANDB_SETTING]
```

There are two modes avaliable for training and testing the model, namely: `train` and `test`.


### Avalilable Models:
1. UNet
2. ENet
